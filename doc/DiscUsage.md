# DiscUsage

Class for disc space information.

## Properties
Name | Type | Description | Notes
---- | ---- | ----------- | -----
**used_size** | **int** | Application used disc space. | 
**total_size** | **int** | Total disc space. | 



## Example
```php
$discUsage = new DiscUsage
{
    UsedSize = 1048576,
    TotalSize = 3145728
};
```
