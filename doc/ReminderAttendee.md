# ReminderAttendee

Defines an \&quot;Attendee\&quot; within a alarm.

## Properties
Name | Type | Description | Notes
---- | ---- | ----------- | -----
**address** | **string** | Contains the email address. | 



## Example
```php
$reminderAttendee = new ReminderAttendee
{
    Address = "attendee@aspose.com"
};
```
